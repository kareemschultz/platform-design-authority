# Platform Design Authority - Agent Instructions

`CLAUDE.md` is the repository-wide, agent-neutral contributor contract. Every automated contributor MUST read and obey it. Directory `AGENTS.md` files may add guidance but cannot weaken authority, lifecycle, mandatory lookup, validation, or immutable-audit rules.

After governed edits run registry generation, `scripts/validate_docs.py`, and freshness checks. Corrections to `reviews/FABLE*` belong in a separate registration or disposition.
