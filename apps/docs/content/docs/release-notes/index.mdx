- Frozen install, formatting, typecheck, workspace and browser tests, production builds, Node health, registry freshness, prohibited-surface scan, dependency audit, PostgreSQL verification, and Docker image checks in CI.

## Compatibility notes

- Bun is the preferred prototype runtime; the Hono server retains a Node 24 fallback.
- oRPC remains pinned to the verified published package set recorded in the implementation conflict log.
- Deep Windows checkout paths can exceed Turbopack's path limit during the Fumadocs build; use a shorter local path or the Linux CI result.
- PostgreSQL extensions remain governed by ADR-0024 and are absent unless specifically allowed.

## Upgrade and support status

No compatibility or support window is promised for this prototype checkpoint. Dependency changes require current technology evidence, a reviewed diff, frozen-lock validation, runtime/build tests, and updated lessons. Customer-facing release notes begin only after an approved product release exists.
